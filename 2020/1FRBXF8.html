3.1.2. 鉴权机制

sequenceDiagram
    客户端 ->>+ 服务器: 输入用户名和密码
    服务器 ->>+ Mysql数据库: 根据用户和密码查询
    Mysql数据库 -->>- 服务器: 返回结果集
    alt 查询到了
        服务器 ->> 服务器: 根据结果集生成Token
        服务器 -->> 客户端: 放在Cookie返回给客户端，登录成功
        客户端 ->> 客户端: 存储token
    else 未查询到
        服务器 -->>- 客户端: 登录失败
    end
    客户端 ->>+ 服务器: 其他操作请求
    服务器 ->> 服务器: 从请求中获取cookie中的token
    服务器 ->>+ Redis数据库: 从数据库中查询token
    Redis数据库 -->>- 服务器: 返回结果
    alt 查询到了
        服务器 ->> 服务器: 解析Token中携带的用户id
        服务器 ->> 服务器: 其他操作
        服务器 -->> 客户端: 操作完成
    else 未查询到
        服务器 ->>- 客户端: 请求失败,Token过期或不存在,发送重定向请求
    end

3.2. 用户模块

3.2.1. 登录

sequenceDiagram
    登录界面->>+服务器: 输入用户名和密码
    服务器->>+数据库: 根据用户名和密码查询
        数据库-->>服务器: 返回结果集
    alt 查询到了
            服务器-->>登录界面: 登录成功
        rect rgba(0, 0, 255, .1)
            登录界面->>用户界面: 跳转到用户界面
        end
    else 未查询到
        服务器-->>-登录界面: 登录失败
    end

3.2.2. 注册

sequenceDiagram
    注册界面->>+服务器: 输入注册信息
    服务器->>+数据库: 插入数据库
    数据库-->>服务器: 返回插入结果
    alt 未触发唯一约束
        服务器-->>注册界面: 注册成功
        rect rgba(0,0,255,.1)
            注册界面->>登录界面:跳转到登录界面
        end
    else 触发唯一约束
        服务器-->>-注册界面: 注册失败
    end

3.2.3. 修改密码

sequenceDiagram
    修改密码界面->>+服务器: 输入旧密码和新密码
    服务器->>+数据库: 根据用户名和旧密码去更新数据
    数据库-->>服务器: 返回更新结果
    alt 更新成功
            服务器-->>修改密码界面: 修改密码成功
    else 更新失败
        服务器-->>-修改密码界面: 旧密码错误
    end

3.2.4. 银行知识学习（文档）

sequenceDiagram
    学习界面->>+服务器: 用户点击列表中相关文档学习
    服务器->>+数据库: 根据文档id查询资源路径
    数据库->>-服务器: 返回资源路径
    服务器->>服务器: 根据文件路径读取文件
    alt 读取到了
        服务器->>学习界面: 返回资源
        学习界面->>学习界面: 渲染资源
    else 没有读取到
        服务器->>-学习界面: 资源不存在
    end

4. 数据库设计

4.1. 用户和管理员表（user_info）

名称 键名 类型 大小 备注
表记录编号 uuid bigint 主键
用户名 username varchar 20 唯一
密码 password varchar 16 非空、md5加密
手机号码 phonenum varchar 11 唯一、非空
创建时间 createtime timestamp 默认数据库时间
更新时间 updatetime timestamp 更新操作会设置为数据库时间

4.2. 学习文档表（doc_list）

名称 键名 类型 大小 备注
表记录号 dlid bigint 自增主键
文档名 name varchar 16 非空
存储位置 path varchar 非空
文档上传人 uuid bigint 外键
文档学习时长 duration int 非空
创建时间 createtime timestamp 默认数据库时间
更新时间 updatetime timestamp 更新操作会设置为数据库时间

4.3. 文档学习记录表 （learn_record）

名称 键名 类型 大小 备注
表记录号 lrid bigint 主键自增
文档表记录号 dlid bigint 外键、非空、同步删除、联合唯一
学习人 uuid bigint 外键、非空、同步删除、联合唯一
学习时长 duration int 非空、默认0
创建时间 createtime timestamp 默认数据库时间
更新时间 updatetime timestamp 更新操作会设置为数据库时间

5. 接口说明

5.1. 4.1 用户接口

5.1.1. 登录接口

  • 路径：/user/login
  • 方法：POST
  • 参数：
    • username：用户名
    • password：密码
  • 返回：
    • uiid：用户编号

5.1.2. 注册接口

  • 路径：/user/register
  • 方法：POST
  • 参数：
    • username：用户名
    • password：密码
    • phonenum：手机号码

5.1.3. 获取个人资料

  • 路径：/user/getmyprofile
  • 方法：GET
  • 参数：token验证
  • 返回：
    • username：用户名
    • uiid：用户编号
    • phonenum：手机号码

5.1.4. 获取个人文档学习记录

  • 路径：/user/getlearnrecord
  • 方法：GET
  • 参数：
    • token验证
  • 返回：
    • lrid：学习记录号
    • name：文档名称
    • doc_duration：文档学习时间
    • duration：文档已学时间
    • updatetime：更新时间
    • createtime：创建时间

5.1.5. 分页获取文档列表

  • 路径：/user/getdoclist
  • 方法：GET
  • 参数
    • token验证
    • page：页数
  • 返回：
    • name：文档名称
    • dlid：文档编号
    • duration：时长
    • createtime：创建时间

5.1.6. 文档学习记录时间更新接口

  • 路径：/user/updatedoctime
  • 方法：POST
  • 参数：
    • token验证
    • lrid：学习记录号
    • newtime：需要更新的时间

5.1.7. 添加学习记录接口

  • 路径：/user/addlearnrecord
  • 方法：POST
  • 参数：
    • token验证
    • dlid：文档编号
